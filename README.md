# Recipe book
A simple website to store recipes. It uses a simple API to store, modify and remove recipes.

![Website Preview](/screenshots/recipe_book_list.png)

## Key Feature
- Live search
- Categories
- Favourite recipes

## Technologies
- FrontEnd: 
  - HTML
  - CSS
  - ReactJS
  - Redux
- BackEnd:
  - NodeJS
  - Express
  - SQLite3

## Setup
1. Development:
   * `npm install`
   * `npm start`
   * `npm run dev`
2. Live:
   * `npm build`
   * `npm start`
